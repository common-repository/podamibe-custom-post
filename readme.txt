Custom post plugin to create custom post types.

== Description ==
<strong>Podamibe Custom Post</strong> is a <strong>Free WordPress plugin</strong> to manage posts in various custom post type.

== Installation ==
1. Unzip podamibe-custom-post.zip
2. Upload all the files to the /wp-content/plugins/podamibe-custom-post
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. For customizing the plugin's settings, click on podamibe-custom-post menu in Wordpress left admin menu.

== Frequently Asked Questions ==
= What does this plugin do? =
This plugin provides to create different custom post type where you can add posts from backend.
